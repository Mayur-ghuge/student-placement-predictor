About Project

📌 Project Overview

Student Placement Prediction System is an AI-powered web application that predicts a student's placement chances based on academic performance, coding skills, communication skills, projects, internships and backlogs.

🛠 Technologies Used

  • Python
  • Flask
  • MySQL
  • Machine Learning
  • HTML
  • CSS
  • JavaScript

🤖 Machine Learning Model

Random Forest Classifier is used to analyze student data and predict placement probability.

✨ Features

  • Placement Prediction
  • Prediction History
  • Analytics Dashboard
  • MySQL Database Storage
  • Interactive User Interface
